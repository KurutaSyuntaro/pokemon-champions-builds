    # 構築済みパーティー

対戦で使用するパーティーの記録を保存します。

## ファイル命名規則

`{レギュレーション}_{フォーマット}_{軸ポケモン}.md`

- **レギュレーション**: `m-a`, `m-b` など小文字ハイフン区切り
- **フォーマット**: `singles` / `doubles`
- **軸ポケモン**: 構築の中心となるポケモン（英語・小文字ハイフン区切り）

例: `m-a_doubles_mega-charizard-y.md`

## テンプレート

新しい構築を作成する際は [_TEMPLATE.md](_TEMPLATE.md) をコピーしてください。
